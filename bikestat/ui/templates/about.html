about bikestat

bikestat is a django app that pulls in data from Capital Bikeshare's page of trip history data. I'm an avid bikeshare user and a data junkie so I thought might be fun to do something interesting to highlight how cool it is that they're sharing the data.

This site based on free software and free data and you can have the code that runs it, it's free software too. See github.com/dchud/bikestat for the code.
