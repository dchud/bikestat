about bikestat

bikestat pulls in data from Capital Bikeshare's page of trip history data. I'm an avid bikeshare user and an info junkie so I thought might be fun to do something interesting to highlight how cool it is that they're sharing the data.

This site is based on free data, and you can have the code that runs it, too, it's free software. See github.com/dchud/bikestat for the code. bikestat is a django app using PostgreSQL, D3, and bootstrap. If you have suggestions for enhancements or fixes please feel free to fork the code on github, send pull requests, and file tickets.

For any other questions or comments, I'm @dchud on twitter.
